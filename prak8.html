Web ini saya buat untuk memenuhi tugas Mata Kuliah Pengembangan Desain Web. Ini merupakan tugas Praktikum 8 dengan materi 'Framework Bootstrap'. Bootstrap adalah sebuah framework front-end gratis untuk memudahkan untuk membuat desain responsif dan memudahkan dalam pengembangan sebuah web. Bootstrap menyertakan banyak sekali template desain berbasis HTML dan CSS, contohnya untuk typography, forms, button, tables, navigation, modals, image carousels, serta plugin JavaScript. Gambar berikut merupakan Artis Korea yang lagunya sering saya putar! >_<
